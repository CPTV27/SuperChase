## THE AGENT SYSTEM

### Pipeline Stages
1. **QUEUED** - Task awaiting execution
2. **BUILDING** - Agent actively creating deliverable
3. **REVIEW** - Quality assurance check
4. **PUBLISH** - Deployment to platforms
5. **ARCHIVE** - Logged with metrics

### Agent Roles

**PLANNER (Claude - Strategic)**
- Creates tasks based on goals
- Prioritizes work across ventures
- Adjusts strategy intensity

**BUILDER (Claude Code - Execution)**
- Executes one task per session
- Reads brand.json for voice
- Creates deliverables in /outputs/

**REVIEWER (Claude - QA)**
- Checks voice, accuracy, requirements
- PASS → /approved/
- FAIL → revision notes

**PUBLISHER (Manual/API)**
- Deploys to platforms
- Records URLs and timestamps

**ARCHIVIST (Claude - Logging)**
- Updates metrics
- Creates archive records
- Extracts learnings

### Venture Councils

Each venture has specialized advisors:

**Scan2Plan Council:**
- LIBRARIAN (Research) - Market intelligence
- AUDITOR (Skeptic) - Margin enforcement
- ARCHITECT (Strategy) - Synthesis

**Big Muddy Council:**
- STORYTELLER - Southern Gothic narrative
- LOCAL SEO - GBP optimization
