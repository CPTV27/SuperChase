## STANDARD KICKOFF FORMAT

Copy and customize this for EVERY new Claude/AI thread:

```
## PROJECT CONTEXT

**Project:** [NAME]
**Code:** [VENTURE-CODE]
**Bible:** [PATH_TO_PROJECT_BIBLE]

## VENTURE

**Business:** [Scan2Plan | Big Muddy | Studio C | Tuthill | CPTV | Utopia | Feed Farm]
**Brand Guide:** /superchase/clients/[VENTURE]/brand.json
**Voice:** [One-line voice description]

## CURRENT STATE

**Phase:** [Kickoff | Production | Review | Delivery]
**Last Action:** [What was just completed]
**Blockers:** [Any known issues]

## THIS SESSION

**Intent:** [Single sentence — what do you need from this thread]
**Files Needed:** [Specific paths this session requires]
**Output:** [Where deliverables should go]

## SUCCESS CRITERIA

This thread is successful when:
- [ ] [Specific outcome 1]
- [ ] [Specific outcome 2]

## CONSTRAINTS

- [Time limit, budget, technical restrictions]
- [Brand voice requirements]
- [Approval needed before X]
```
