## Common Issues to Flag

### Name Variations (Incorrect)
- ❌ Big Muddy Inn (missing "& Blues Room")
- ❌ The Big Muddy Inn
- ❌ Big Muddy B&B
- ❌ Big Muddy Hotel

### Address Variations (Incorrect)
- ❌ 411 N Commerce St (abbreviation inconsistency)
- ❌ 411 Commerce Street (missing "North")
- ❌ Any address in "Under-the-Hill" without street number

### Phone Issues
- ❌ Old phone number
- ❌ Fax number listed as primary
- ❌ No phone number listed

## Audit Process

### Step 1: Search Each Directory
1. Go to directory website
2. Search "Big Muddy Inn Natchez"
3. If found: check all fields against correct NAP
4. If not found: note as "Not Listed"

### Step 2: Document Issues
For each listing, note:
- Name shown
- Address shown
- Phone shown
- Website shown
- Any other incorrect info

### Step 3: Claim or Correct
- If unclaimed: Claim the listing (may require verification)
- If incorrect: Submit correction through directory's edit process
- If duplicate: Flag for removal

### Step 4: Track Changes
- Note date correction submitted
- Check back in 2-4 weeks to verify update
- Some directories take 30-60 days to update

## Priority Corrections

### Google Business Profile (Do First)
1. Log into Google Business Profile Manager
2. Verify business name matches exactly
3. Check address, phone, website
4. Add/update photos
5. Set correct categories (primary: Boutique Hotel)
6. Add attributes (live music, parking, etc.)

### TripAdvisor (High Impact)
1. Go to TripAdvisor Management Center
2. Claim listing if not already
3. Update all contact info
4. Add professional photos
5. Respond to recent reviews

### Yelp (Often Incorrect)
1. Claim Yelp Business Page
2. Verify contact info
3. Add business hours
4. Upload photos
5. Monitor for user edits
