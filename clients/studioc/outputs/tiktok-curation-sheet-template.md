## Automation Notes

**TikTok Export Method:**
1. Go to Profile → Liked Videos
2. For each video: long press → Copy Link
3. Paste into sheet, VA fills metadata manually
4. OR use TikTok data export (Settings → Privacy → Download Data)

**Spotify Track Finding:**
1. Search sound name + artist in Spotify
2. If not found, try Shazam on the TikTok video
3. If still not found, note as "TikTok Original Only"

**Cross-posting:**
- Newsletter publishes Sunday evening
- Twitter thread posts Monday morning
- Playlist updates rolling (as found)
